About HexClaw

A full-stack open source AI Agent ecosystem built by the hexagon-codes team.

Team

HexClaw is developed and maintained by the hexagon-codes team. We are committed to building a local-first, privacy-preserving, full-stack open source AI Agent toolchain so everyone can use large language models safely and efficiently.

Open Source Ecosystem

The project uses a layered architecture across 7 repositories, each independently usable:

  • toolkit — Infrastructure
  • ai-core — LLM adapters
  • hexagon — Agent framework
  • hexclaw — Backend service
  • hexclaw-desktop — Desktop client
  • hexclaw-ui — Web client
  • hexagon-ui — Agent console

Technical Architecture

HexClaw is built on our in-house Hexagon Agent engine, using a Tauri + Go stack. The desktop app is built with Tauri for cross-platform native deployment. The backend Sidecar, written in Go, powers Agent orchestration, MCP integration, knowledge base, workflows, and more. All data is stored locally, and Ollama enables fully offline operation.
