Keyboard Shortcuts

Global shortcuts and common in-page commands.

Global Shortcuts

ShortcutAction
⌘+Shift+HQuick chat, which opens the floating window globally
⌘+KCommand palette for fast search and navigation
⌘+,Open settings
⌘+NCreate a new session

Chat Page

ShortcutAction
EnterSend the current message
Shift+EnterInsert a line break
⌘+FSearch conversations
⌘+EExport the current session
EscStop generation

Workflow Canvas

ShortcutAction
⌘+SSave the workflow
DeleteDelete the selected node
⌘+ZUndo
⌘+Shift+ZRedo

Navigation

ShortcutAction
⌘+1Chat
⌘+2Agents
⌘+3Knowledge
⌘+4Automation
⌘+5Connections
⌘+6Integration
⌘+7Logs
⌘+8Settings
⌘+BCollapse or expand the sidebar

System Tray

InteractionAction
Left-click the tray iconOpen or focus the main window
Right-click the tray iconOpen the tray menu with chat, logs, settings, and quit options

Command Palette

Press ⌘ + K to open the command palette. It supports the following actions:

Navigation Commands

CommandDescription
ChatGo to the Chat page
AgentGo to Agent Management
KnowledgeGo to Knowledge (Documents / Memory)
AutomationGo to Automation (Jobs / Webhooks / Workflows)
ConnectionsGo to Channels & Accounts and Data Connectors
IntegrationGo to Skills / MCP / Prompts
LogsGo to Real-time Logs
SettingsGo to the Settings page

Action Commands

CommandDescription
New ConversationCreate a new session and navigate to Chat
Toggle ThemeSwitch between light and dark mode

Type keywords to quickly filter available commands.

Tip: On macOS use ⌘. On Windows and Linux the equivalent is usually Ctrl.
