يېڭىلىنىش خاتىرىسى

ھېشى AI ئۈستەلئۈستى خېرىدارىنىڭ نەشر يېڭىلىنىش خاتىرىسى.

v0.4.0 (2026-04-18)

  • رەسىم، سىن ۋە ئاۋاز ھاسىل قىلىش Composer لىرى قوشۇلدى.
  • ھاسىل قىلىنغان مېدىيا نەتىجىلىرىنى كۆرۈشۈشكە داۋاملىق ساقلاش قوللاندى.
  • رەسىم چۈشۈرۈش macOS WebView چەكلىمىسىدىن ئايرىلىپ، يەرلىك ساقلاش دىئالوگى ئارقىلىق ئىشلىدى.
  • كۆرۈشۈش داۋاملاشتۇرۇشتىكى 403 ۋە قايتا قوزغاتقاندىن كېيىن رەسىم كۆرۈنمەسلىك مەسىلىسى تۈزىتىلدى.
  • Ollama ئالدىن قىزىتىش، Provider ئىزدەش ۋە خاتالىق يوللىرىغا قوشۇمچە سىناقلار قوشۇلدى.

v0.3.8 (2026-04-14)

  • ZhipuAI CogView-4 رەسىم ھاسىل قىلىش ۋە CogVideoX-2 سىن ھاسىل قىلىش قوللاندى.
  • قورال ناملىرىنىڭ خەلقئارالاشتۇرۇلۇشى ياخشىلاندى.
  • كۆڭۈلدىكى مودېل تاللاش، كۆپ قۇرلۇق ئەستە تۇتۇش ۋە Ollama چۈشۈرۈش جەريانى ئەلالاشتۇرۇلدى.

v0.3.7 (2026-04-13)

  • ئويلىنىش ۋاقتى، مودېل ۋە Agent metadata لىرى يەرلىك ساقلاشقا داۋاملىق يېزىلىدۇ.
  • كۆرۈشۈش نامى ئاپتوماتىك ھاسىل قىلىنىدۇ.
  • قورال ئىجرا خاتالىقى LLM غا تولۇق قايتۇرۇلۇپ، ئۆزىنى تۈزىتىش ئايلانمىسى كۈچەيتىلدى.
  • كۆپلىگەن ئارقا ئۇچ ۋە ئالدى ئۇچ سىناقلىرى تۈزىتىلدى.

v0.3.6 (2026-04-11)

  • ئېقىملىق چىقىرىش جەريانىدا ئىشلەتكۈچىنىڭ ئۈستىگە سىيرىلىشى ئۈزۈلمەيدىغان بولدى.
  • Agent ۋە IM كۆرۈنمە يۈزىدىكى بىر قانچە ئىشلىتىش مەسىلىسى تۈزىتىلدى.

v0.3.5 (2026-04-06)

  • Ollama قوزغالغاندا يەرلىك مودېلنى ئالدىن يۈكلەش قوشۇلدى.
  • تارىخىي ئۇچۇرلاردىكى think بەلگىسىنى ئايرىپ بىر تەرەپ قىلىش كۈچەيتىلدى.
  • MCP filesystem نىڭ home مۇندەرىجىسى macOS، Windows ۋە Linux قا ماسلاشتۇرۇلدى.

v0.3.0 - v0.3.4

  • MCP بازىرى، قورال چاقىرىش زەنجىرى ۋە Ollama مودېل باشقۇرۇش كۆرۈنمە يۈزى تولۇقلاندى.
  • نەشر نومۇرى كۆرسىتىش ۋە سىستېما ھالىتى ئۇچۇرلىرى بىرلىككە كەلتۈرۈلدى.

v0.2.x

  • ئېقىملىق reasoning كۆرسىتىش، Ollama يەرلىك مودېل، MCP بازار ۋە قۇرۇلمىلىق خاتىرە ئىقتىدارلىرى پەيدىن-پەي تولۇقلاندى.
  • Playwright ۋە TypeScript سىناقلىرى ئارقىلىق كۆرۈشۈش، بىلىم ئامبىرى، ئەستە تۇتۇش، Skill، MCP ۋە خاتىرە يوللىرى قاپلاندى.
