v0.4.0 (2026-04-18)

  • 图像/سىن/ئاۋاز生成三件套: ImageGenComposer / VideoGenComposer / VoiceChatComposer، 按مودېل能力自动ئالماشتۇرۇش، 生成结果داۋاملاشتۇرۇش到كۆرۈشۈش
  • ئەسلىيساقلاشسۆھبەت框: 图像چۈشۈرۈش走 Tauri dialog.save() + Rust 命令写盘، 绕过 WKWebView چۈشۈرۈش限制، كۆڭۈلدىكىھۆججەت名 HexClaw-yyyymmdd-hhmmss-XXXX.{ext} 避免同名冲突
  • كۆرۈشۈشداۋاملاشتۇرۇش 403 ئوڭشاش: appendSessionMessage/Batch 把 user_id 放到 URL query (كەينى ئۇچ sessionUserIDFromRequest 只读 query) ; ئوڭشاشرەسىم ھاسىللاشئۇچۇرقايتا قوزغىتىشكۆرۈشۈش后不显示的根因
  • 生成ھالەت顶栏ئالماشتۇرۇش: 右上角 text-only chip بولىدۇ从图像/سىن/ئاۋازمودېل切回 chat مودېل، 不打断كۆرۈشۈش
  • Apple HIG 打磨: 三件套 Composer padding 12→20، gap 8→14، 字号 11→13–14، 圆角 10/16، 0.5px 细边框، focus 0 0 0 3px 蓝光环; 左下角موتوربەلگە改ئۈچۈن Hexagon engine
  • Ollama 预热 tag 4 级匹配: tag 精确 → base 去 tag → provider.selectedModelId → downloaded[0]
  • Provider key 硬编码 Bug 治理: 生成服务按 BaseURL 特征查找 Provider، 解决中文 Provider name (如"ZhipuAI AI") 匹配不到的سوئال
  • 63 条错误路径سىناش: cron-errors / skills-errors / mcp-errors; 结构性防护سىناش防止裸 apiPost/apiGet 绕过 sessionPost 包装器
  • كەينى ئۇچ hexclaw v0.4.0: 批量ئۇچۇر事务写入 / genstore 原子 tmp+Rename / videogen singleflight+LRU / imagegen/videogen/voicechat داۋاملاشتۇرۇش到 file_path

v0.3.8 (2026-04-14)

  • رەسىم生成: قوللاشZhipuAI CogView-4، رەسىم内嵌 data URI 永不过期، ai-core يېڭى قوشۇلغان ImageProvider 接口
  • سىن ھاسىللاش: قوللاشZhipuAI CogVideoX-2، 异步任务轮询، 封面内嵌 + سىن内联播放، ai-core يېڭى قوشۇلغان VideoProvider 接口
  • قورال名 i18n: 15 个内置قورال中英文映射 (code_exec → 代码沙箱 قاتارلىق) ، 未映射的 MCP/ئىختىيارىيقورال自动 fallback 原名
  • كۆڭۈلدىكىمودېلئوڭشاش: 普通سۆھبەت不再被كۆڭۈلدىكى ئاجانت مودېل偏好قاپلاش، ئاجانت ھالەت显示 ئاجانت 名字来源标注

  • Ollama چۈشۈرۈشئەلالاشتۇرۇش: 大مودېل不再超时 (ResponseHeaderTimeout 替代全局 Timeout) ، چۈشۈرۈش状态中文化 (获取مودېلئۇچۇر/校验ھۆججەت完整性/写入مودېلھۆججەت)
  • 智能路由 UI: 关闭时隐藏策略下拉框، 开启后才显示
  • 飞书 Reaction: 从社交 emoji (💪☕🔥) 改ئۈچۈنمەنا精准的 Typing ⌨️ / THINKING 🤔
