يېڭىلىنىش خاتىرىسى

ھېشى AI ئۈستەلئۈستى خېرىدارىنىڭ نەشر يېڭىلىنىش خاتىرىسى.

v0.4.8 (2026-06-28)

  • تەرتىپلىك مەزمۇن بۆلەكلىرى: ئويلىنىش، تېكىست ۋە قورال چاقىرىش سەۋەب-نەتىجە تەرتىپىدە ئارىلاش كۆرسىتىلىدۇ؛ قورال چاقىرىش كارتىسى ۋە تارماق Agent تاختىسى قوشۇلدى.
  • ئۇچۇر يوللاش تېخىمۇ ئەگىشىشچان: ئىشلەتكۈچى ئۇچۇرى دەرھال ئېكرانغا چىقىدۇ، بىلىم ئامبىرى ئىزدىشى (Auto-RAG) تەرىپىدىن توسۇلمايدۇ.
  • IM قاناللىرىنى Agent غا باغلاش: قانال بىۋاسىتە Agent غا باغلىنىدۇ، مودېل Agent خاسلىقى سۈپىتىدە بىرتۇتاش باشقۇرۇلىدۇ.
  • بىلىم ئامبىرى: RAG ئىزدەش پارامېتىرلىرىنى كۆرۈنۈشلۈك سەپلەش، ۋاقىت دائىرىسى بويىچە سۈزۈش ۋە بەتلەش.
  • ئۇزۇلۇكسىز ۋەزىپە تۈرى قوشۇلدى: مەزگىللەر بويىچە ئۈزۈلمەي ئىلگىرىلەيدىغان ئۇزۇن ۋەزىپىلەرنى قوللايدۇ.
  • ئۈستەلئۈستىدە ئۈن بىلەن ئوقۇش (TTS، ھەقسىز edge-tts) سۈكۈتتە قوزغىتىلىدۇ، مەغلۇپ بولسا توختىتىپ قايتا سىناشقا بولىدۇ.
  • خىزمەت ئېقىمى مەغلۇپ بولسا داۋاملاشتۇرۇشقا بولىدۇ، تاماملانغان تۈگۈنلەرنى قايتا ئىشلىتىدۇ.
  • ئەسلەك سىستېمىسى بىرلا FileMemory غا بىرتۇتاشلاندى، ئۇزۇن مۇددەتلىك ئەسلەك «ئۇزۇن مۇددەتلىك ئەسلەك» بېتىدە مەركەزلەشتۈرۈلدى.
  • ئاستىدىكى hexclaw v0.4.8 كە يېڭىلاندى.

v0.4.7 (2026-06-26)

  • خاتالىق ئىنكاسى ۋە ياقتۇرۇش/ياقتۇرماسلىق باھاسى ساقلىنىدۇ: سۆھبەت ئالماشتۇرغان ياكى قايتا يۈكلىگەندىن كېيىنمۇ يوقالمايدۇ، باھانى يەنە بېرىشكە بولىدۇ.
  • ئۈن بىلەن ئوقۇش (TTS) مەغلۇپ بولغاندا ئېنىق ئەسكەرتىش چىقىرىدۇ، سۈكۈتتە ئىنكاسسىز قالمايدۇ.
  • ناچارلىشىش ئۈزگۈچى: مودېلنىڭ ناچارلاشقان چىقىرىشىنى بايقاپ توسىدۇ، سۆھبەتنى بۇلغاشتىن ساقلايدۇ.
  • ھۆججەتنى دېتال ئىچىدە چۈشۈرۈش ۋە كۆرۈش: چۈشۈرۈش سىستېمىنىڭ ئەسلى ساقلاش سۆزلىشىش رامكىسىدىن ئۆتىدۇ؛ PDF تەھلىلى ئارقا ئۇچقا يۆتكەلدى، ئالدى ئۇچ pdfjs نى يەنە قاچىلىمايدۇ.
  • ئۆز-ئارا تەسىر تۈزىتىشلىرى: يېڭى سۆھبەتتە سىيرىش يولباشچىسى خاتا كۆرۈنمەيدۇ، كىرگۈزگۈچ بىرىكمە ھالىتىدە Enter خاتا يوللىمايدۇ، سۆھبەت مودېلى سۆھبەتلەر ئارىسىدا ئارىلاشمايدۇ.
  • ئاستىدىكى hexclaw v0.4.7 كە يېڭىلاندى.

v0.4.6 (2026-06-23)

  • ئۇلىنىش مەركىزىگە MCP قوشۇلدى؛ MCP مۇلازىمېتىرىنىڭ env ئىسپاتلىرى ئاساسىي ئاچقۇچ بىلەن شىفىرلىنىپ ساقلىنىدۇ، API ئىنكاسىدا مەخپىيلەشتۈرۈلىدۇ.
  • max-turns مۇلايىم بىر تەرەپ قىلىش: قورال چاقىرىش قېتىم چېكىگە يەتكەندە خاتالىق قايتۇرماي، قىسمەن نەتىجە ۋە «داۋاملاشتۇرۇش» ئەسكەرتىشى بېرىلىدۇ؛ كۆڭۈلدىكى چەك 25 كە ئۆستى.
  • ھۆججەت ئەسلەك مەسىلىسى تۈزىتىلدى: يېڭى ئەسلەك قايتا قوزغىتىشسىز دەرھال كۈچكە ئىگە.
  • ئۈستەلئۈستى ئۇقتۇرۇشلىرىغا مەنبە بەلگىسى (cron / IM) قوشۇلدى.
  • ئاستىدىكى hexclaw v0.4.6 (Hexagon v0.5.3) ۋە ماھارەت بازىرى v0.0.5 كە يېڭىلاندى.

v0.4.5 (2026-06-22)

  • ئۇلىنىش مەركىزى: قانال ۋە ھېسابات + سانلىق مەلۇمات ئۇلىغۇچلىرى؛ قانال كارتىلىرى IM ۋە ئېلخەت (SMTP/IMAP) نى قايتا ئىشلىتىدۇ، يەرلىكتە شىفىرلىنىدۇ.
  • سانلىق مەلۇمات ئۇلىغۇچلىرى (GitHub / Notion): token پەقەت ئوقۇشلۇق ئۇلىنىدۇ، ھەقىقىي سىناق ۋە بايلىق كۆرۈش؛ token ئارقا ئۇچتا شىفىرلىنىپ ساقلىنىدۇ.
  • سىزىقلىق ئىش ئېقىمى تەھرىرلىگۈچ: باسقۇچ قوشۇش / ئۆچۈرۈش / تەرتىپلەش / تەھرىرلەش / ساقلاش / سىناق ئىجرا، ھەر بىر تۈگۈن ھالىتى دەرھال قايتۇرۇلىدۇ.
  • كۆڭۈلدىكى ياردەمچى كىشىلىكى (SOUL) نى تەھرىرلەش: ~/.hexclaw/SOUL.md نى ئوقۇش/يېزىش.
  • Prompt ئامبىرى، رەسىمنى سۆرەپ/چاپلاپ vision غا بېرىش، ⌘K بۇيرۇق تاختىسى؛ ئۈچ تىل (RTL) تولۇق قاپلاندى.
  • Windows نەشرى پەقەت NSIS (.exe) چىقىرىدۇ، .msi چىقىرىلمايدۇ؛ Windows غا Ollama يۈكلەنمەيدۇ.

v0.4.2 (2026-06-16)

  • ماھارەت تەپسىلاتى SKILL.md نى پەقەت ئوقۇش ھالىتىدە كۆرسىتىدۇ؛ ئاتالغۇ «ماھارەت» كە بىرلىكتى.
  • بىلىم ئامبىرى ھۆججەت كۆرگۈچىسى Markdown نى رەندەرلەيدۇ؛ كود بۆلىكىدە hover دا كۆچۈرۈش.
  • Webhook قوزغىتىلمىغان ھالەتنى بىر تەرەپ قىلىدۇ؛ كۆرۈشۈش يېزىش API كېلىشىمى بىرلىكتى.
  • pandoc / typst نى Tauri sidecar غا ئۆزگەرتىش؛ Windows نەشر قۇرۇلۇشى تۈزىتىلدى.

v0.4.0 (2026-04-30)

  • چوڭ قايتا قۇرۇش · بىرلىككەن تېكىست رامكىسى: ئايرىم رەسىم/سىن ھاسىل قىلىش composer لىرى ئېلىۋېتىلىپ، بارلىق يول بىرلا ChatInput غا بىرلەشتۈرۈلدى (ChatGPT / Claude / Gemini ئەندىزىسى).
  • ئۇيغۇرچە (ug) RTL قوللىشى: 1330 i18n ئاچقۇچى خەنزۇچە بىلەن 1:1 ماس؛ <html dir=rtl lang=ug> ئاپتوماتىك بەلگىلىنىدۇ، ئارىلاش يېزىق يۆنىلىشى توغرا، code / pre مەجبۇرىي LTR.
  • Apple HIG بەش ئۆلچەمدە توغرىلاندى: يان سىزىق ≤1.5px، سايە ≤0.12، ئېنىق ئالماشتۇرۇش خاسلىقى، ساپ Apple خەت نۇسخىسى.
  • مېدىيا چۈشۈرۈش / كۆرۈش: رەسىم/سىن ئۈچۈن بىرلىككەن چۈشۈرۈش كۇنۇپكىسى (0.85 شەپپاپلىقتا داۋاملىق كۆرۈنىدۇ)، چېكىلسە تولۇق ئېكران lightbox.
  • ئارقا سۇپىدا ھاسىل قىلىش → يەرلىككە ساقلاش → URL پايدىلىنىش: نەتىجىلەر generated/ غا ساقلىنىدۇ، ئالدى ئۇچ مەڭگۈ ۋاقتى ئۆتمەيدىغان يەرلىك URL نى ئالدىن ئىشلىتىدۇ.
  • يېڭى Interactive بۆلەكلىرى (Buttons / Select / Approval / Card / Block) + message.interactive كېلىشىمى.
  • ئارقا ئۇچ hexclaw v0.4.0: Feature Flag / Skill Pipeline / مودېل ئىقتىدار بايقاش / ۋەقە توشۇش v1 / مودېل دەرۋازىسى v1 / قورال ھايات دەۋرى v2.
