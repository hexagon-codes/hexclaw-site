ئىشلىتىش شەرتلىرى

ھېشى AI نى ئىشلىتىشىڭىز تۆۋەندىكى شەرتلەرگە قوشۇلغانلىقىڭىزنى بىلدۈرىدۇ.

ئوچۇق كودلۇق كېلىشىم

ھېشى AI Apache-2.0 ئوچۇق كودلۇق كېلىشىمى بىلەن تارقىتىلىدۇ. سىز بۇ يۇمشاق دېتالنى ئەركىن ئىشلىتىپ، ئۆزگەرتىپ ۋە تارقىتالايسىز، ئەمما شۇ كېلىشىمدىكى مۇناسىۋەتلىك شەرتلەرگە ئەمەل قىلىشىڭىز كېرەك.

ئەسلى ھالىتىدە تەمىنلىنىدۇ

بۇ يۇمشاق دېتال «ئەسلى ھالىتىدە» تەمىنلىنىدۇ، ئاشكارا ياكى يوشۇرۇن ھېچقانداق كاپالەت بېرىلمەيدۇ؛ بۇ سودا قىلىشقا ماسلىقى، مەلۇم بىر مەقسەتكە ماسلىقى ۋە ھوقۇق دەخلىسى يوقلۇقى قاتارلىقلارنىمۇ ئۆز ئىچىگە ئالىدۇ.

API Key بىخەتەرلىك

ئىشلەتكۈچى API Key نى ساقلاش ۋە ئۇنىڭ بىخەتەرلىكىگە ئۆزى مەسئۇل. API Key نى باشقىلار بىلەن ئورتاقلاشماڭ ياكى ئوچۇق ئامبارغا يوللىماڭ. Key ئاشكارىلىنىشىدىن كېلىپ چىققان زىيانغا ئىشلەتكۈچى ئۆزى مەسئۇل بولىدۇ.

ئۈچىنچى تەرەپ AI مۇلازىمەتلىرى

OpenAI، Anthropic، DeepSeek قاتارلىق ئۈچىنچى تەرەپ AI مۇلازىمەتلىرىنى ئىشلەتكەندە، شۇ مۇلازىمەت تەمىنلىگۈچىلەرنىڭ ئىشلىتىش شەرتلىرى ۋە شەخسىيەت سىياسىتىگە ئەمەل قىلىشىڭىز كېرەك. سانلىق مەلۇمات يولى، ئىشلەتكۈچى كونتروللىرى ۋە مۇھىم ئەسكەرتىشلەرنى ئۈچىنچى تەرەپ AI مۇلازىمىتى ۋە سانلىق مەلۇمات چۈشەندۈرۈشىدىن كۆرۈڭ.

مەسئۇلىيەتسىزلىك بايانى

hexagon-codes ۋە ھېشى AI تۈرىگە تۆھپە قوشقۇچىلار بۇ يۇمشاق دېتالنى ئىشلىتىشتىن كېلىپ چىققان ھېچقانداق بىۋاسىتە، ۋاسىتىلىك، قوشۇمچە ياكى نەتىجىلىك زىيانغا مەسئۇل بولمايدۇ. ئىشلەتكۈچى بۇ يۇمشاق دېتالنى ئىشلىتىش ئارقىلىق قىلغان قارار ۋە ھەرىكەتلىرىگە ئۆزى مەسئۇل.

بۇ مەسئۇلىيەت تەقسىماتى پەقەت قوللىنىلىدىغان قانۇن رۇخسەت قىلغان دائىرىدە كۈچكە ئىگە. قانۇن بويىچە چەتكە قېقىشقا بولمايدىغان ياكى چەكلەشكە بولمايدىغان مەسئۇلىيەت ۋە ئىشلەتكۈچىنىڭ قانۇندا بەلگىلەنگەن ھوقۇقى تەسىرگە ئۇچرىمايدۇ.
