شەخسىيەت سىياسىتى

ھېشى AI شەخسىيىتىڭىزنى ھۆرمەت قىلىدۇ، بارلىق سانلىق-مەلۇمات يەرلىكتە بىر تەرەپ قىلىنىدۇ.

سانلىق-مەلۇمات ساقلاش

بارلىق سانلىق-مەلۇمات ئىشلەتكۈچىنىڭ يەرلىك ئۈسكۈنىسىدە ساقلىنىدۇ. ھېشى AI سۆھبەتلىرىڭىزنى، تەڭشەكلىرىڭىزنى ياكى ھېچقانداق شەخسىي سانلىق-مەلۇماتنى بىزنىڭ مۇلازىمېتىرغا يوللىمايدۇ.

API Key بىخەتەرلىك

API Key مەشغۇلات سىستېمىسىنىڭ ئاچقۇچ زەنجىرىدە (Keychain / Credential Manager) بىخەتەر ساقلىنىدۇ، تەڭشەك ھۆججىتىدە ئاددىي تېكىست شەكلىدە ساقلانمايدۇ.

تور ئالاقىسى

ھېشى AI پەقەت سىز سەپلىگەن AI API ئاخىرقى نۇقتىلىرى بىلەن ئالاقىلىشىدۇ. سىز مۇناسىۋەتلىك AI مۇلازىمەت تەمىنلىگۈچىنى ئۆزىڭىز سەپلىمىسىڭىز، بىز سانلىق-مەلۇماتلىرىڭىزنى ئۈچىنچى تەرەپ مۇلازىمېتىرلىرىغا يوللىمايمىز.

تولۇق تورسىز ھالەت

Ollama قاتارلىق يەرلىك مودېللارنى ئىشلەتكەندە، ھېشى AI تولۇق تورسىز ئىجرا بولالايدۇ، بارلىق سانلىق-مەلۇمات ئۈسكۈنىڭىزدىن ئايرىلمايدۇ.

سانلىق-مەلۇمات توپلاش ۋە ئىز قوغلاش

بىز ئىشلەتكۈچى سانلىق-مەلۇماتىنى توپلىمايمىز، ئىشلەتكۈچى ھەرىكىتىنى ئىز قوغلىمايمىز، ئېلان كۆرسەتمەيمىز. ھېشى AI ئوچۇق كودلۇق يۇمشاق دېتال، شۇڭا مەنبە كودنى خالىغان ۋاقىتتا تەكشۈرۈپ بۇ ۋەدىلەرنى دەلىللىيەلەيسىز.

ئوچۇق كودلۇق ۋە ئاشكارىلىق

ھېشى AI Apache-2.0 ئوچۇق كودلۇق كېلىشىمى بىلەن تارقىتىلىدۇ، تولۇق مەنبە كودنى GitHub دا كۆرگىلى بولىدۇ. جامائەتنىڭ كود تەكشۈرۈشى ۋە تۆھپە قوشۇشىنى قارشى ئالىمىز.
