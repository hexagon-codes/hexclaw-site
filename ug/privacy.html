باش بەتمەھسۇلات ئىقتىدارىتېز قول سېلىشئىشلىتىشئەھۋالئوچۇق كودلۇقئېكوسىستېماھۆججەتلەرGitHub

شەخسىيەت سىياسىتى

ھېشى AI 尊重سىزنىڭشەخسىيەت، بارلىقسانلىق-مەلۇمات均在本地处理.

سانلىق-مەلۇماتساقلاش

بارلىقسانلىق-مەلۇماتساقلانغانئىشلەتكۈچى本地设备. ھېشى AI 不بولىدۇ将سىزنىڭسۆھبەت، تەڭشەكياكى任何个人سانلىق-مەلۇمات上传到بىزنىڭمۇلازىمېتىر.

API Key بىخەتەرلىك

API Key ئارقىلىق操作سىستېما密钥链 (Keychain / Credential Manager) بىخەتەرلىكساقلاش، 不بولىدۇ以明文形式ساقلاش在تەڭشەشھۆججەت中.

网络通信

ھېشى AI 仅بىلەنسىزتەڭشەش的 AI API 端点通信. بىز不بولىدۇ向第三方مۇلازىمېتىر发送سىزنىڭسانلىق-مەلۇمات، 除非سىز主动تەڭشەش了相应的 AI مۇلازىمەت تەمىنلىگۈچى.

完全离线ھالەت

ئىشلىتىش Ollama قاتارلىقيەرلىك مودېل时، ھېشى AI بولىدۇ完全离线ئىجرا قىلىش، بارلىقسانلىق-مەلۇمات均不离开سىزنىڭ设备.

سانلىق-مەلۇمات收集بىلەن追踪

بىز不收集ئىشلەتكۈچىسانلىق-مەلۇمات، 不追踪ئىشلەتكۈچى行ئۈچۈن، 不投放广告. ھېشى AI ئۈچۈنئوچۇق كودلۇق软件، سىزبولىدۇ随时审计代码以验证上述承诺.

ئوچۇق كودلۇقبىلەن透明

ھېشى AI ئىشلىتىش Apache-2.0 ئوچۇق كودلۇقكېلىشىم发布، 完整مەنبە كودىبولىدۇ在 GitHub 上查阅. بىز欢迎جامائەت参بىلەن代码审计بىلەنتۆھپە.
