ھېشى AI ھەققىدە

hexagon-codes ئەترىتى ياسىغان تولۇق ستاك ئوچۇق كودلۇق AI Agent ئېكوسىستېمىسى.

ئەترەت تونۇشتۇرۇشى

ھېشى AI (HexClaw) hexagon-codes ئەترىتى تەرىپىدىن ئېچىلىپ ۋە ئاسرىلىدۇ. بىز يەرلىك ئالدىنلىق، سانلىق-مەلۇمات شەخسىي، تولۇق ستاك ئوچۇق كودلۇق AI Agent قورال زەنجىرىنى قۇرۇپ، ھەممە ئادەمگە چوڭ تىل مودېللىرىنى بىخەتەر ۋە ئۈنۈملۈك ئىشلىتىش ئىمكانى بېرىشنى نىشان قىلىمىز.

ئوچۇق كودلۇق ئېكوسىستېما

تۈر 7 ئامبارغا ئايرىلغان قەۋەتلىك قۇرۇلما ئىشلىتىدۇ، ھەر بىر قەۋەتنى ئايرىم ئىشلىتىشكە بولىدۇ:

  • toolkit — ئاساسىي ئۇل ئەسلىھە
  • ai-core — LLM ماسلاشتۇرۇش
  • hexagon — ئاجانت رامكا
  • hexclaw — ئارقا ئۇچ مۇلازىمىتى
  • hexclaw-desktop — ئۈستەلئۈستى خېرىدار
  • hexclaw-ui — Web خېرىدار
  • hexagon-ui — Agent كۆزىتىش تاختىسى

تېخنىكىلىق قۇرۇلما

ھېشى AI ئۆزىمىز تەتقىق قىلغان Hexagon Agent ماتورىغا ئاساسلىنىدۇ، Tauri + Go تېخنىكا ستاكىنى ئىشلىتىدۇ. ئۈستەلئۈستى خېرىدارى Tauri ئارقىلىق كۆپ سۇپىلىق يەرلىك ئەپ قىلىپ قۇرۇلغان؛ ئارقا ئۇچ Sidecar بولسا Go ئارقىلىق Agent پىلانلاش، MCP بىرلەشتۈرۈش، بىلىم ئامبىرى ۋە ئىش ئېقىمى قاتارلىق يادرولۇق ئىقتىدارلارنى ئىجرا قىلىدۇ. سانلىق-مەلۇماتلارنىڭ ھەممىسى يەرلىكتە ساقلىنىدۇ، Ollama ئارقىلىق تولۇق تورسىز ئىجرا قىلىشنى قوللايدۇ.

بىز بىلەن ئالاقىلىشىش

سودا ھەمكارلىقى ياكى تېخنىكىلىق مەسلىھەت ئۈچۈن تۆۋەندىكى يوللار ئارقىلىق ئالاقىلىشىڭ:

  • سودا مەسلىھەتى: ai@hexclaw.net
  • سوئال-پىكىر: support@hexclaw.net
