مەھسۇلات ئىقتىدارلىرى

بۇ پەقەت سۆھبەت خېرىدارىلا ئەمەس، بەلكى AI Agent نى ئېچىش ۋە ئىجرا قىلىش ئۈچۈن تولۇق سۇپا.

يەرلىك ئورنىتىش، سانلىق-مەلۇمات شەخسىي

Sidecar قۇرۇلمىسى يەرلىك كومپيۇتېردا ئىجرا بولىدۇ؛ API Key مودېل تەمىنلىگۈچىگە بىۋاسىتە ئۇلىنىدۇ، ئوتتۇرىدا ۋاكالەتچى يوق.

ئۆزىمىز تەتقىق قىلغان Agent ماتورى

ReAct ۋە Plan-and-Execute دىن ئىبارەت ئىككى خىل پىلانلاش ئۇسۇلى، بايانىي ئېنىقلىما ۋە قوراللارنى ئاپتوماتىك ۋاقىتلاشتۇرۇش.

MCP قورال كېلىشىمى

Model Context Protocol نى يەرلىك قوللايدۇ؛ ھۆججەت، ساندان ۋە API لارنى تېز ئۇلاپ ئىشلىتىشكە بولىدۇ.

كۆپ مودېلنى بىرلىككە كەلتۈرۈپ ماسلاشتۇرۇش

OpenAI / Claude / DeepSeek نى بىرلىككە كەلتۈرۈپ ئوراپ، ئىجرا ۋاقتىدا قىزىق ئالماشتۇرۇشنى قوللايدۇ.

پۈتۈن زەنجىرنى كۆزىتىش

ئىجرا زەنجىرىنى ئىزلاش، پىلانلاشنى قايتا كۆرۈش، Tool Call خاتىرىسى ۋە Token سەرپىياتىنى تەھلىل قىلىش.

ئۈچ سۇپىدا يەرلىك ئىجرا

Tauri v2 + Rust ئارقىلىق macOS / Windows / Linux دا يەرلىك، يېنىك ۋە تېز ئىجرا بولىدۇ.

2

API Key نى تولدۇرۇش

Key پەقەت يەرلىكتە ساقلىنىدۇ، ھېچقانداق ئوتتۇرا مۇلازىمەتتىن ئۆتمەي مودېل تەمىنلىگۈچىگە بىۋاسىتە ئۇلىنىدۇ.
3

سۆھبەتلا بۇيرۇق
