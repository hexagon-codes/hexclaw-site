ئىش ئېقىمى تاختىسى

بولىدۇ视化拖拽تەرتىپلەش AI ئىش ئېقىمى، قوللاش DAG ئىجرا موتورى.

概述

ئىش ئېقىمى تاختىسى是ھېشى AI 的 无代码 AI تەرتىپلەشقورال. ئارقىلىق拖拽节点ۋە连线، 构建复杂的 AI 处理管线.

节点类型

节点چۈشەندۈرۈش典型用途
ئاجانتئىجرا قىلىش AI سۆھبىتى任务内容生成، سانلىق-مەلۇماتئانالىز، 代码编写
Toolچاقىرىش MCP قورالAPI 请求، ھۆججەت操作، سانلىق-مەلۇمات库查询
Condition条件分支判断根据结果走不同路径
Output输出最终结果ساقلاشھۆججەت، 发送通知، قايتىشسانلىق-مەلۇمات

قۇرۇشئىش ئېقىمى

  1. كىرىش ئىش ئېقىمى تاختىسى بەت
  2. 从左侧面板拖拽节点到画布
  3. چېكىش节点间的ئۇلىنىش点، 拖拽قۇرۇش连线
  4. چېكىش节点تەڭشەش参数 (تاللاش ئاجانت، تەڭشەكئەسكەرتىش词قاتارلىق)
  5. ساقلاشئىش ئېقىمى

模板库

内置多种预制ئىش ئېقىمى模板، بولىدۇتېز باشلاش:

  • ھۆججەتلەر翻译流水线 — 翻译 → 校对 → 润色
  • 代码审查 — بىخەتەرلىكتەكشۈرۈش → 性能ئانالىز → 最佳实践
  • سانلىق-مەلۇمات处理 — 提取 → 清洗 → ئانالىز → بولىدۇ视化
  • 内容创作 — 大纲 → 初稿 → تەھرىرلەش → 终稿

ئىجرا قىلىشئىش ئېقىمى

  1. ساقلاش后چېكىش "ئىجرا قىلىش" كۇنۇپكا
  2. ئىش ئېقىمى按 DAG (有向无环图) 拓扑顺序ئىجرا قىلىش
  3. 实时显示ھەر بىر节点的ئىجرا قىلىش状态: pending → running → completed / failed
  4. ئىجرا قىلىش完成后كۆرۈش输出结果
ئەسكەرتىش: ئىش ئېقىمىئىجرا قىلىشقوللاش状态轮询، بولىدۇ在ئىجرا قىلىش过程中离开بەت، 稍后回来كۆرۈش结果.

ئىش ئېقىمى验证

چېكىش"ئىجرا قىلىش"前، ھېشى AI بولىدۇ自动验证ئىش ئېقىمى:

  • 空画布تەكشۈرۈش — 至少ئېھتىياج一个节点
  • 孤立节点تەكشۈرۈش — بارلىق节点必须ئارقىلىق连线ئۇلىنىش
  • 循环检测 — 不允许存在环形依赖 (DAG 约束)

验证失败时بولىدۇ在顶部显示错误ئەسكەرتىش، 需ئوڭشاش后才能ئىجرا قىلىش.

ساقلاشبىلەن加载

قورال栏提供ئىش ئېقىمىباشقۇرۇشئىقتىدار:

  • ساقلاش — 输入名称ۋە描述، ساقلاش当前画布ئۈچۈنئىش ئېقىمى
  • 加载 — 从已ساقلاش的ئىش ئېقىمىتىزىملىك中تاللاش加载到画布
  • ئۆچۈرۈش — 在加载تىزىملىك中ئۆچۈرۈش不ئېھتىياج的ئىش ئېقىمى

ئىجرا قىلىش状态

ئىجرا قىلىشئىش ئېقىمى时، ھەر بىر节点بولىدۇ实时显示ئىجرا قىلىش状态:

  • ⏳ قاتارلىق待中 — 尚未ئىجرا قىلىش
  • 🔄 ئىجرا قىلىش中 — 正在ئىجرا قىلىش (蓝色边框 + 旋转动画)
  • ✅ 已完成 — ئىجرا قىلىش成功 (绿色边框)
  • ❌ 失败 — ئىجرا قىلىش出错 (红色边框)

ئىجرا قىلىش完成后، 底部结果面板显示输出内容ياكى错误ئۇچۇر.
