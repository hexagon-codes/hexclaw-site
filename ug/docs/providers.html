مودېل تەمىنلىگۈچى

AI مۇلازىمەت تەمىنلىگۈچى، مودېل تىزىملىكى، API Key ۋە يەرلىك مودېللارنى باشقۇرۇش.

قوللايدىغان تەمىنلىگۈچىلەر

ھېشى AI OpenAI، Anthropic Claude، DeepSeek، Google Gemini، Qwen، Doubao، Ollama ۋە OpenAI API بىلەن ماس كېلىدىغان خاس endpoint لارنى قوللايدۇ.

API Key ساقلاش

API Key يەرلىك سىستېما ئاچقۇچ زەنجىرىدە ساقلىنىدۇ. ئۇچۇرلار بىزنىڭ مۇلازىمېتىرغا يوللانمايدۇ؛ تەلەپ بىۋاسىتە سىز تاللىغان تەمىنلىگۈچىگە ئەۋەتىلىدۇ.

مودېل تاللاش

  1. تەمىنلىگۈچىنى قوشۇڭ.
  2. ئۇلىنىش سىنىقىنى ئىجرا قىلىڭ.
  3. كۆڭۈلدىكى سۆھبەت مودېلىنى تاللاڭ.
  4. Agent ياكى ئىش ئېقىمىغا ئايرىم مودېل بەلگىلىيەلەيسىز.

Ollama يەرلىك مودېل

Ollama قوشۇلغاندا base URL ئادەتتە http://127.0.0.1:11434 بولىدۇ. API Key بوش قالدۇرۇلىدۇ، چۈشۈرۈلگەن مودېللار تىزىملىكتىن تاللىنىدۇ.

خاس تەمىنلىگۈچى

OpenAI API فورماتىغا ماس مۇلازىمەتلەر ئۈچۈن base URL، API Key ۋە مودېل نامىنى قولدا كىرگۈزۈڭ. بۇ خاس gateway، ئىچكى proxy ياكى enterprise endpoint لارغا ماس كېلىدۇ.
