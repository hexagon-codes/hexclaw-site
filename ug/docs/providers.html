قوشۇشمۇلازىمەت تەمىنلىگۈچى

  1. كىرىش تەڭشەك → مودېل مۇلازىمەت تەمىنلىگۈچى
  2. چېكىش "قوشۇشمۇلازىمەت تەمىنلىگۈچى"
  3. تاللاش类型 (OpenAI / DeepSeek / ...)
  4. تولدۇرۇش API Key (بىخەتەرلىكساقلانغان Tauri 本地密钥库中)
  5. (ئىختىيارىي) 修改 Base URL (适用于代理ياكىشەخسىي部署)
  6. ساقلاش

Ollama يەرلىك ئورنىتىش

ئىشلىتىش Ollama بولىدۇ完全离线ئىجرا قىلىش AI مودېل: 

# ئورنىتىش Ollama
brew install ollama    # macOS
# ياكى访问 ollama.com چۈشۈرۈش

# قوزغىتىش并چۈشۈرۈشمودېل
ollama serve
ollama pull llama3.1
ollama pull qwen2.5
ollama pull deepseek-r1

在ھېشى AI 中قوشۇش Ollama مۇلازىمەت تەمىنلىگۈچى، Base URL 填 http://localhost:11434/v1، ئېھتىياج يوق API Key.

API Key بىخەتەرلىك

بىخەتەرلىك: بارلىق API Key ئارقىلىق Tauri 的بىخەتەرلىكساقلاش机制ساقلاش在操作سىستېما密钥链中، 不بولىدۇ以明文写入磁盘.

سىناشئۇلىنىش

ھەر بىر Provider 卡片上有 ⚡ سىناشكۇنۇپكا، چېكىش后بولىدۇ验证:

  • كەينى ئۇچموتور是否ئىجرا قىلىش
  • API تەڭشەش是否بولىدۇ访问

سىناش结果بولىدۇ内联显示在 Provider 卡片上 (绿色 ✓ ياكى红色 ✕) .

自动مودېلتاللاش

مودېل下拉تىزىملىك的第一个选项是 Auto (自动تاللاش، 故障ئالماشتۇرۇش) . تاللاش后:

  • 优先ئىشلىتىشكۆڭۈلدىكى Provider 的كۆڭۈلدىكىمودېل
  • 如果请求失败، 自动ئالماشتۇرۇش到下一个بولىدۇ用 Provider
  • 适合ئېھتىياج高بولىدۇ用性的ئەھۋال
