تېز باشلاش

5 مىنۇتتا ئورنىتىش، سەپلىمە ۋە تۇنجى سۆھبەتنى تاماملاش.

سىستېما تەلىپى

سۇپىلارئەڭ تۆۋەن تەلەپ
macOSmacOS 12 Monterey ياكى ئۇنىڭدىن يۇقىرى، Apple Silicon / Intel
WindowsWindows 10 1809+، x64
LinuxUbuntu 20.04+ ياكى Fedora 36+، x64

ئورنىتىش

macOS بىر قۇرلۇق سكرىپت

curl -fsSL https://raw.githubusercontent.com/hexagon-codes/hexclaw-desktop/main/install.sh | bash

Homebrew

brew tap hexagon-codes/tap && brew install --cask hexclaw

ئورنىتىش بولىقى

GitHub Releases دىن v0.4.9 نىڭ macOS، Windows ۋە Linux بولاقلىرىنى چۈشۈرۈڭ.

  • macOS: .dmg
  • Windows: .exe（NSIS）
  • Linux: .deb، .AppImage ياكى .rpm

تۇنجى قوزغىتىش

  1. AI مۇلازىمەت تەمىنلىگۈچىنى تاللاپ API Key نى تولدۇرۇڭ.
  2. كۆڭۈلدىكى رول ۋە مودېلنى جەزملەشتۈرۈڭ.
  3. ئۇلىنىش سىنىقىدىن كېيىن سۆھبەت بېتىگە كىرىڭ.

مودېل سەپلىمىسى

تەڭشەكتىكى مودېل تەمىنلىگۈچى بېتىدىن OpenAI، Claude، DeepSeek، Gemini، Qwen، Doubao، Ollama ياكى OpenAI API بىلەن ماس كېلىدىغان خاس تەمىنلىگۈچىنى قوشالايسىز.

ئەسكەرتىش: Ollama يەرلىك مودېل ئىشلىتىلگەندە API Key تەلەپ قىلىنمايدۇ، سانلىق-مەلۇمات ئۈسكۈنىڭىزدىن ئايرىلمايدۇ.

ئەپ قۇرۇلمىسى

  • ئالدى ئۇچ: Vue 3 + TypeScript
  • ئۈستەلئۈستى قېپى: Tauri v2 + Rust
  • AI ماتورى: Go Sidecar، Agent پىلانلاش ۋە قورال چاقىرىش
  • سانلىق-مەلۇمات: يەرلىك SQLite ساقلاش

كۆرۈنمە يۈز قۇرۇلمىسى

ئەپ قوزغالغاندا كۆڭۈلدىن Chat بېتىگە كىرىدۇ. يان بالداق ئۈچ گۇرۇپپىغا ئايرىلغان: قۇرۇش(Agent، بىلىم ئامبىرى، ئاپتوماتلاشتۇرۇش)، ئۇلىنىش(ئۇلىنىش مەركىزى، بىرلەشتۈرۈش)، سىستېما(خاتىرە، تەڭشەك). ⌘+K بۇيرۇق تاختىسى ياكى ⌘+سان بىلەن خالىغان بەتكە سەكرىگىلى بولىدۇ.

مەسىلە چىقسا

  1. مۇلازىمەت تەمىنلىگۈچى سەپلىمىسىدىن ئۇلىنىشنى سىناڭ.
  2. خاتىرە بېتىدە Engine قوزغالغانلىقىنى ۋە 16060 پورتى ئىشلىتىلمىگەنلىكىنى تەكشۈرۈڭ.
  3. 401، 403، بوش بەت ياكى MCP ئۇلىنىش مەسىلىسى بولسا كۆپ كۆرۈلىدىغان سوئاللار بېتىنى كۆرۈڭ.
