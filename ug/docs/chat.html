ھېشى AI ھۆججەتلەر

سۆھبەتبىلەنكۆرۈشۈش

多مودېلسۆھبەت، كۆرۈشۈشباشقۇرۇش، Artifact، ئۇچۇرئىزدەشبىلەنچىقىرىش.

基本سۆھبەت

كىرىش سۆھبەت بەت即بولىدۇباشلاشسۆھبەت. ھېشى AI قوللاش:

  • ئېقىپ چىقىرىش — AI 回复实时逐字显示
  • Markdown 渲染 — 完整قوللاش Markdown 语法، 代码块自动语法高亮
  • 多مودېلئالماشتۇرۇش — 同一كۆرۈشۈش中بولىدۇ随时ئالماشتۇرۇش不同مۇلازىمەت تەمىنلىگۈچىۋەمودېل
  • ئۇچۇر操作 — كۆچۈرۈش، تەھرىرلەش، 重新生成回复

ھۆججەتبىلەن多模态

كىرگۈزۈش رامكىسىقوللاش发送多种类型的附件 (取决于所选مودېل能力) :

类型قوللاشمودېل
رەسىمGPT-4o، Claude 全系، Gemini 全系، Qwen-VL، Doubao Vision، LLaVA
سىنGemini 2.5 / 2.0 يۈرۈشلۈك
ئاۋازGemini 2.5 / 2.0 يۈرۈشلۈك
ھۆججەتبارلىق文本مودېل (自动提取内容)
ئەسكەرتىش: كىرگۈزۈش رامكىسىبولىدۇ根据当前مودېل的 capabilities 自动显示/隐藏对应的附件كۇنۇپكا.

كۆرۈشۈشباشقۇرۇش

  • يېڭىكۆرۈشۈش — چېكىش左侧 "+" كۇنۇپكاقۇرۇش
  • كۆرۈشۈش历史 — 左侧面板显示بارلىق历史كۆرۈشۈش، چېكىشئالماشتۇرۇش
  • ئۆچۈرۈشكۆرۈشۈش — 右键ياكى滑动操作ئۆچۈرۈش
  • كۆرۈشۈشداۋاملاشتۇرۇش — بارلىقئۇچۇرساقلانغان本地 SQLite سانلىق-مەلۇمات库中

Artifact ئالدىن كۆرۈش

当 AI 输出代码块时، ھېشى AI بولىدۇ自动提取ئۈچۈن Artifact:

  • قوللاش类型: code (代码) ، html (网页ئالدىن كۆرۈش) ، markdown، file
  • 代码 Artifact 带语法高亮ۋەبىر كۇنۇپكاكۆچۈرۈش
  • HTML Artifact بولىدۇ直接在侧面板中实时ئالدىن كۆرۈش

ئۇچۇرئىزدەش

ئىشلىتىش ئىزدەشسۆھبەت ئىقتىدار (⌘+F) بولىدۇ跨بارلىقكۆرۈشۈشئىزدەشئۇچۇر内容.

ئۇچۇرچىقىرىش

چېكىشكۆرۈشۈشتىزىملىك中的 چىقىرىش كۇنۇپكا، قوللاش三种格式:

  • JSON — 完整结构化سانلىق-مەلۇمات، 适合زاپاسۋەئەكىرىش
  • Markdown — بولىدۇ读性好، 适合归档ۋە分享
  • PDF — 适合正式ھۆججەتلەر输出

@ 提及

在كىرگۈزۈش رامكىسى中输入 @ بولىدۇ弹出提及تىزىملىك، 快速引用 ئاجانت ياكى团队成员加入سۆھبەت.

سۆھبەتھالەت

ھېشى AI 提供三种سۆھبەتھالەت، ئارقىلىق顶部بەلگەئالماشتۇرۇش:

ھالەت用途چۈشەندۈرۈش
سۆھبەت日常问答标准سۆھبەت، 直接بىلەن LLM سۆھبەت
ئاجانت任务ئىجرا قىلىشئاجانت 自主规划، چاقىرىشقورال، 多步ئىجرا قىلىش
深度研究调研报告多步ئىزدەش→ئانالىز→综合→生成报告، 显示 4 阶段进度面板
ئەسكەرتىش: 从 ئاجانت باشقۇرۇش页چېكىشرول的"سۆھبەت"كۇنۇپكا، بولىدۇ自动ئالماشتۇرۇش到 ئاجانت ھالەت并以رول名作ئۈچۈنكۆرۈشۈش标题.

ھۆججەتلەر解析

سۆھبەت时بولىدۇ直接上传ھۆججەتلەرھۆججەت، ھېشى AI بولىدۇ自动提取文本内容作ئۈچۈنمۇھىت发送给 LLM:

格式قوللاش类型
PDF.pdf — 提取全部بەت文本، 显示页数
Word.docx — 提取正文内容
Excel.xlsx / .csv — 按 Sheet 提取ئۈچۈن表格文本
纯文本.txt / .md / .json

文本超过 50,000 字符时بولىدۇ自动截断并ئەسكەرتىش.

سۆھبەت参数

چېكىشسۆھبەت区顶部قورال栏的 ⚙️ 参数 كۇنۇپكا، بولىدۇ调整当前كۆرۈشۈش的参数:

  • Temperature (0 ~ 2) — 越高回复越有创意، 越低越确定
  • Max Tokens (256 ~ 128,000) — 单次回复的最大长度

参数仅对当前كۆرۈشۈش生效، 不影响其他كۆرۈشۈش.

ئۇچۇر操作

右键تىزىملىك

在ئۇچۇر气泡上右键 (ياكى长按) ، 弹出مۇھىتتىزىملىك:

  • كۆچۈرۈش — كۆچۈرۈشئۇچۇر内容到剪贴板
  • 重试 — ئۆچۈرۈش当前回复并重新生成
  • تەھرىرلەش — 修改已发送的ئىشلەتكۈچىئۇچۇر并重新获取回复
  • ئۆچۈرۈش — ئۆچۈرۈش该ئۇچۇر (同步ئۆچۈرۈشسانلىق-مەلۇمات库خاتىرە)

كۆرۈشۈش重命名

在左侧كۆرۈشۈشتىزىملىك中، 双击كۆرۈشۈش标题بولىدۇ直接重命名. يەنەبولىدۇئارقىلىق右键تىزىملىكتاللاش"重命名".

Token 计数

顶部قورال栏显示当前كۆرۈشۈش的预估 Token 数 (如 ~1.2k tok) ، 帮助评估مۇھىت窗口ئىشلىتىش情况.

ئاۋاز输入

چېكىشكىرگۈزۈش رامكىسى右侧的 🎤 麦克风كۇنۇپكا، قوزغىتىشئاۋاز识别 (STT) . 录音完成后自动转ئۈچۈن文本تولدۇرۇشكىرگۈزۈش رامكىسى. ئېھتىياج浏览器قوللاش Web Speech API ياكىتەڭشەش外部 STT 服务.
