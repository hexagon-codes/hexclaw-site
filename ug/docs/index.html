ھېشى AI ئىشلىتىشھۆججەتلەر

قاپلاشھېشى AI ئۈستەلئۈستى ئۇچى的ئورنىتىش، تەڭشەشبىلەنيادرولۇق ئىقتىدارلار; 底层رامكاۋەموتورھۆججەتلەر仍在各自仓库单独维护.

8 个核心主题ئورنىتىش到ئالىي ئىقتىدارلار面向مەھسۇلاتئىشلەتكۈچى
تېز قول سېلىش
完成ئورنىتىش، تەڭشەش API Key، 并بىرىنچى ئۇچۇر يوللاشئۇچۇر.
سۆھبەتبىلەنكۆرۈشۈش
بىلىپ ئېلىشسۆھبەت، كۆرۈشۈشباشقۇرۇش، Artifact ئالدىن كۆرۈشبىلەنئۇچۇرچىقىرىش.
ئاجانت باشقۇرۇش
قۇرۇشرول، تەڭشەشئەسكەرتىش词بىلەنقورال، 并组织多 ئاجانت 协作.
ئىش ئېقىمى تاختىسى
用بولىدۇ视化节点تەرتىپلەش任务، 并ئىجرا قىلىش DAG ئىش ئېقىمى.
مودېل مۇلازىمەت تەمىنلىگۈچى
باشقۇرۇشبىرنەچچە AI مۇلازىمەت تەمىنلىگۈچى، مودېلتىزىملىكبىلەن本地 API Key.
ئالىي ئىقتىدارلار
كۆرۈش MCP، بىلىم ئامبىرى، ئەستە تۇتۇش، ۋاقىتلىق ۋەزىپەبىلەنبىخەتەرلىك能力.
تېز كۇنۇپكا پايدىلىنىش
查阅ئومۇمىي تېز كۇنۇپكاۋەبەت内常用操作组合.
كۆپ كۆرۈلىدىغان سوئاللار
快速定位ئورنىتىش، ئۇلىنىشۋەئىشلىتىش中的كۆپ كۆرۈلىدىغان سوئاللار.
