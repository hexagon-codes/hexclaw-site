ئاجانت باشقۇرۇش

ئىختىيارىي تەڭشەك قۇرۇش AI رول، تەڭشەشسىستېمائەسكەرتىش词ۋەقورال، كۆپ خىل قوزغىتىش ئاجانت بولىدۇ议.

什么是 ئاجانت

ئاجانت 是ھېشى AI 中的 AI رول. ھەر بىر ئاجانت 拥有独立的:

  • سىستېمائەسكەرتىش词 (System Prompt) — 定义رول的行ئۈچۈنۋە专业能力
  • مودېلتاللاش — بولىدۇ指定ئىشلىتىش哪个 AI مودېل
  • 温度参数 — 控制创造力 vs 确定性
  • قورال集 — 分配 MCP قورال给特定 ئاجانت
  • ماھارەت集 — 启用已ئورنىتىش的ماھارەت

قۇرۇش ئاجانت

  1. كىرىش ئاجانت باشقۇرۇش بەت
  2. چېكىش "قۇرۇش ئاجانت"
  3. 填写名称، 描述، سىستېمائەسكەرتىش词
  4. (ئىختىيارىي) تاللاش绑定مودېل، تەڭشەك温度، 分配قورالۋەماھارەت
  5. ساقلاش
مىسال: قۇرۇش一个 "ئالدى ئۇچ专家" ئاجانت، System Prompt 设ئۈچۈن "你是一位资深ئالدى ئۇچقۇرۇلۇش师، 精通 Vue 3، TypeScript، TailwindCSS..."، 温度设ئۈچۈن 0.3.

ئاجانت 属性

属性چۈشەندۈرۈش
name英文标识符، 唯一
display_name显示名称
description简短描述
system_promptسىستېمائەسكەرتىش词، 定义رول行ئۈچۈن
model绑定的مودېل (留空则ئىشلىتىشكۆڭۈلدىكىمودېل)
temperature生成温度 0~2 (كۆڭۈلدىكى 0.7)
tools[]بولىدۇ用的 MCP قورالتىزىملىك
skills[]启用的ماھارەتتىزىملىك
statusactive / inactive

ئىشلىتىش ئاجانت سۆھبەت

在سۆھبەتبەت، چېكىشكىرگۈزۈش رامكىسى上方的 ئاجانت تاللاش器ئالماشتۇرۇشرول. 选中的 ئاجانت 的سىستېمائەسكەرتىش词بولىدۇ自动注入到سۆھبەت中.

多 ئاجانت بولىدۇ议

ھېشى AI قوللاش 多 ئاجانت 协作مۇلاھىزە ھالەت:

  1. 在 ئاجانت باشقۇرۇشبەتچېكىش "ئاجانت بولىدۇ议"
  2. تاللاش参بىلەنمۇلاھىزە的بىرنەچچە ئاجانت
  3. 输入议题، بىرنەچچە ئاجانت بولىدۇ轮流发言، مۇلاھىزە، 总结

ماس كېلىدىغان ئەھۋاللار: 技术方案评审، 多角度ئانالىزسوئال، 头脑风暴.

رول详情

在رولتىزىملىك中چېكىش任意رول卡片، بولىدۇ展开كۆرۈش完整ئۇچۇر:

  • 背景设定 — ئاجانت 的رول背景ۋە专长描述
  • قورالتىزىملىك — 该رولبولىدۇچاقىرىش的قورال/ماھارەت
  • 约束条件 — ئاجانت ئىجرا قىلىش任务时需遵守的限制

تېز باشلاشسۆھبەت

在رول卡片上چېكىش"سۆھبەت"كۇنۇپكا:

  1. 自动يېڭى一个以رول名命名的كۆرۈشۈش (如"سانلىق-مەلۇماتئانالىز师")
  2. سۆھبەتھالەت自动ئالماشتۇرۇشئۈچۈن ئاجانت
  3. 后续再次چېكىش同一رول، بولىدۇ定位到已有的كۆرۈشۈش而非重复يېڭى

ئاجانت 路由باشقۇرۇش

在 ئاجانت 路由 بەلگە页باشقۇرۇش ئاجانت 的مودېل绑定:

  • 注册 — 指定 ئاجانت ئىشلىتىش的 Provider ۋەمودېل
  • تەھرىرلەش — چېكىش ✏️ 修改已注册 ئاجانت 的显示名، Provider، مودېل
  • 注销 — 移除 ئاجانت 路由绑定
ئەسكەرتىش: 注册ۋەتەھرىرلەش时، 名称، Provider، مودېل三个字段ھەممىسى不能ئۈچۈن空.

رول定义参考

ئاجانت رول在كەينى ئۇچ hexclaw.yaml 中定义، يەنەبولىدۇئارقىلىق API 动态注册: 

roles:
  - name: researcher
    title: 研究助手
    goal: 深度调研指定主题并生成结构化报告
    backstory: 你是一位资深研究员، 擅长ئۇچۇر检索ۋەئانالىز
    tools:
      - web_search
      - file_write
      - summary
    constraints:
      - 引用来源必须标注
      - 报告字数不少于 1000 字
