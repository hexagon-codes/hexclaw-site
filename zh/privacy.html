隐私政策

河蟹 AI 本地优先；使用您配置的云端模型时，必要内容会发送至对应第三方 Provider。

数据存储

对话、设置与本地索引默认存储在您的设备上。hexagon-codes 不提供用于同步或托管这些数据的官方云端账户服务；当您主动使用第三方 Provider 时，完成该请求所需的输入会按您的配置发送至对应端点。

API Key 安全

桌面端将 API Key 与普通配置分开保存在专用本地存储中；浏览器开发模式只在当前页面内存中临时保留，刷新后需要重新输入。请同时启用设备登录保护和磁盘加密，并避免共享本机账户。

网络通信

河蟹 AI 仅与您配置或启用的网络服务通信。第三方模型可能接收的内容、知识库文本分块、查询文本及用户控制，详见《第三方 AI 服务与数据说明》。

完全离线模式

仅使用 Ollama 等本地模型并关闭其他网络集成时，河蟹 AI 可完全离线运行，相关模型请求不会离开您的设备。

数据收集与追踪

我们不收集用户数据、不追踪用户行为、不投放广告。河蟹 AI 为开源软件，您可随时审计代码以验证上述承诺。

开源与透明

河蟹 AI 采用 Apache-2.0 开源协议发布，完整源代码可在 GitHub 上查阅。我们欢迎社区参与代码审计与贡献。
