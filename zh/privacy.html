隐私政策

河蟹 AI 尊重您的隐私，所有数据均在本地处理。

数据存储

所有数据存储在用户本地设备。河蟹 AI 不会将您的对话、设置或任何个人数据上传到我们的服务器。

API Key 安全

API Key 通过操作系统密钥链（Keychain / Credential Manager）安全存储，不会以明文形式保存在配置文件中。

网络通信

河蟹 AI 仅与您配置的 AI API 端点通信。我们不会向第三方服务器发送您的数据，除非您主动配置了相应的 AI 服务商。

完全离线模式

使用 Ollama 等本地模型时，河蟹 AI 可完全离线运行，所有数据均不离开您的设备。

数据收集与追踪

我们不收集用户数据、不追踪用户行为、不投放广告。河蟹 AI 为开源软件，您可随时审计代码以验证上述承诺。

开源与透明

河蟹 AI 采用 Apache-2.0 开源协议发布，完整源代码可在 GitHub 上查阅。我们欢迎社区参与代码审计与贡献。
