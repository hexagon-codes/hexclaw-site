更新日志

河蟹 AI 桌面端版本更新记录。

v0.4.9 (2026-07-04)

  • 桌面端与后端版本同步至 v0.4.9，内置 sidecar 默认构建自 hexclaw v0.4.9
  • Agent 能力增强：结构化 SOUL 编辑器、技能选择器、高级参数与编辑锁定状态补齐
  • Chat 体验加固：会话分支、删除/导出安全动作、思考时长持久化与纯工具消息兜底
  • 自动化与连接中心：权限控制、定时任务校验、MCP 状态刷新、通道按实例绑定修复
  • 运行时修复：sidecar 优雅退出、记忆行为设置、启动图标内联、重型视图测试超时放宽

v0.4.8 (2026-06-28)

  • 有序内容块渲染：思考、正文与工具调用按因果顺序交错展示，新增工具调用卡片与子 Agent 面板
  • 发送更跟手：用户消息立即上屏，不再被知识库检索（Auto-RAG）往返阻塞
  • IM 通道按 Agent 绑定：通道直接绑定 Agent，模型作为 Agent 属性统一管理
  • 知识库：RAG 检索参数可视化配置、按日期范围筛选与分页
  • 新增持续型任务：支持跨周期持续推进的长任务
  • 桌面端默认启用朗读（TTS，免费 edge-tts），失败可中断重试
  • 工作流运行失败可续接，复用已完成节点
  • 记忆系统统一为单一 FileMemory，长期记忆集中在「长期记忆」页
  • 引擎升级至 hexclaw v0.4.8

v0.4.7 (2026-06-26)

  • 报错回复与点赞/踩反馈持久化：切换会话或重载后不再丢失，反馈仍可再次操作
  • 朗读（TTS）失败时弹出明确提示，不再静默无响应
  • 退化熔断：检测并阻断模型退化输出，避免污染会话
  • 文档应用内下载与预览：下载走系统原生保存对话框；PDF 解析下沉后端，前端不再打包 pdfjs
  • 交互修复：新建会话不再误显滚动导航、输入法组合态回车不误发送、切换会话模型互不串扰
  • 引擎升级至 hexclaw v0.4.7

v0.4.6 (2026-06-23)

  • 连接中心接入 MCP：MCP Server 的 env 凭据（数据库密码、API Key 等）经主密钥静态加密落盘，API 响应一律脱敏
  • max-turns 优雅降级：达到工具调用轮次上限时返回部分结果并在末尾追加「继续」提示，不再向用户抛错；默认轮次上限提升至 25（兜底 50）
  • 文件记忆挂载修复：新增的记忆无需重启即可注入，问答即时生效
  • 桌面通知带来源标识（定时任务 / IM），便于前端映射图标与深链
  • 引擎升级至 hexclaw v0.4.6（Hexagon v0.5.3），技能市场默认分支升至 v0.0.5

v0.4.5 (2026-06-22)

  • 连接中心（二分法）：通道与账号 + 数据连接器；通道卡片复用 IM 实例与邮箱（SMTP/IMAP 按域名自动配），本地加密、「一处存处处引」
  • 数据连接器（GitHub / Notion）：token 只读接入，真实测试连接与浏览资源，token 经后端加密落盘、响应脱敏
  • 线性工作流真实编辑器：新建 / 增删 / 排序 / 编辑步骤、保存、试运行，逐节点状态实时回填
  • 定时任务投递目标选择器：可从连接库下拉选择已配置实例
  • 编辑默认助理人设（SOUL）：读写 ~/.hexclaw/SOUL.md，纯文本编辑，支持恢复内置默认
  • Prompt 库（/ 召唤、$ARGUMENTS 填参）、输入框拖拽 / 粘贴图片走 vision、命令面板 ⌘K、引擎未连接全局降级 banner
  • 通用组件与品牌令牌统一、网格卡 hover 动效；简体中文 / English / 维吾尔语（RTL）三语全覆盖
  • Windows 发版改为仅打 NSIS（.exe），不再提供 .msi；Windows 不再内嵌 Ollama，改为引导单独安装
  • 修复：左下角引擎版本号显示、日志 trace_id 标签重叠、补齐 common.saving 三语键

v0.4.2 (2026-06-16)

  • 导航信息架构重构：Chat 独立置顶 +「构建 / 连接 / 系统」三组分区
  • 技能详情只读查看 SKILL.md，术语统一为「技能」
  • 知识库文档查看器渲染 Markdown；代码块折行、裸围栏轻装饰、hover 复制
  • Webhook 未启用状态处理 + 导航路由同步；会话写接口契约统一

v0.4.1 (2026-06-03)

  • 代码视图渲染下载 + 捆绑 pandoc / typst 渲染链
  • cron 定时任务统一架构 + 端到端 SSE 流式编译

v0.4.0 (2026-04-30)

  • 重大重构 · 统一文本对话框：移除独立的图像 / 视频生成 composer 与生成参数面板，所有路径统一到唯一 ChatInput，对齐 ChatGPT / Claude / Gemini 范式（图像 / 视频走默认参数）
  • 维吾尔语（ug）RTL 支持：1330 条 i18n 与简体中文 1:1 对齐，自动设置 <html dir=rtl lang=ug>，unicode-bidi: plaintext 全局兜底保证中英文 / 数字混排方向正确，code / pre 强制 LTR
  • Apple HIG 五维对齐：边框 ≤1.5px、阴影 alpha ≤0.12、显式过渡属性、纯 Apple 系字体、发光 alpha ≤0.18
  • 媒体下载 / 预览：图片 / 视频统一下载按钮（常驻 0.85 透明度、RTL 安全定位），点击全屏 lightbox 预览
  • 后台生成 → 本地落盘 → URL 引用：生成结果落盘到 generated/，前端优先用永不过期的本地 URL，避免 base64 撑爆数据库
  • 新增 Interactive 通用组件（Buttons / Select / Approval / Card / Block）+ message.interactive 新协议
  • 后端 hexclaw v0.4.0：Feature Flag / Skill Pipeline / 模型能力探测 / 事件传输 v1 / 模型网关 v1 / 工具生命周期 v2

v0.3.8 (2026-04-14)

  • 图片生成：支持智谱 CogView-4，图片内嵌 data URI 永不过期，ai-core 新增 ImageProvider 接口
  • 视频生成：支持智谱 CogVideoX-2，异步任务轮询，封面内嵌 + 视频内联播放，ai-core 新增 VideoProvider 接口
  • 工具名 i18n：15 个内置工具中英文映射（code_exec → 代码沙箱 等），未映射的 MCP/自定义工具自动 fallback 原名
  • 默认模型修复：普通聊天不再被默认 Agent 模型偏好覆盖，Agent 模式显示 Agent 名字来源标注
  • 多行记忆修复：含换行的记忆不再被拆成多条，删除/更新/归档操作支持多行条目
  • Ollama 下载优化：大模型不再超时（ResponseHeaderTimeout 替代全局 Timeout），下载状态中文化（获取模型信息/校验文件完整性/写入模型文件）
  • 智能路由 UI：关闭时隐藏策略下拉框，开启后才显示
  • 飞书 Reaction：从社交 emoji（💪☕🔥）改为语义精准的 Typing ⌨️ / THINKING 🤔
