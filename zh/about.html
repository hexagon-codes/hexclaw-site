关于河蟹 AI

由 hexagon-codes 团队打造的全栈开源 AI Agent 生态。

团队介绍

河蟹 AI（HexClaw）由 hexagon-codes 团队开发维护。我们致力于构建本地优先、数据私有、全栈开源的 AI Agent 工具链，让每个人都能安全、高效地使用大语言模型。

开源生态

项目采用 7 个仓库分层解耦的架构，每一层均可独立使用：

  • toolkit — 基础设施
  • ai-core — LLM 适配
  • hexagon — Agent 框架
  • hexclaw — 后端服务
  • hexclaw-desktop — 桌面客户端
  • hexclaw-ui — Web 客户端
  • hexagon-ui — Agent 观测台

技术架构

河蟹 AI 基于自研 Hexagon Agent 引擎，采用 Tauri + Go 技术栈。桌面端使用 Tauri 构建跨平台原生应用，后端 Sidecar 使用 Go 实现 Agent 编排、MCP 集成、知识库与工作流等核心能力。数据全部存储在本地，支持 Ollama 完全离线运行。
