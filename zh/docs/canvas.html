工作流画布

在「自动化 → 工作流」中，用线性步骤编排多步 AI 任务。

概述

工作流位于 自动化 → 工作流 标签页，是河蟹 AI 的多步 AI 编排工具。它采用线性步骤编排：把一个复杂任务拆成若干顺序步骤，逐步执行并把每步结果传给下一步。

步骤构成

每个步骤是一次 Agent 调用，可分别配置：

配置项说明
角色该步骤使用的智能体 / 人设
提示本步骤的指令，可引用上一步的输出
模型该步骤单独指定的模型
工具本步骤允许调用的工具 / MCP

创建工作流

  1. 进入 自动化 → 工作流 标签页，点击新建
  2. 添加步骤，并按需增删 / 排序
  3. 编辑每个步骤的角色、提示、模型与工具
  4. 保存工作流
  5. 点击试运行，逐节点状态实时回填

执行与状态

运行时，每个步骤按顺序执行，并实时回填状态：

  • ⏳ 等待中 — 尚未执行
  • 🔄 运行中 — 正在执行
  • ✅ 已完成 — 执行成功
  • ❌ 失败 — 执行出错
提示：工作流执行支持状态轮询，可以在执行过程中离开页面，稍后回来查看结果。
