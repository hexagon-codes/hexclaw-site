快捷键参考

全局快捷键和页面内操作快捷键。

全局快捷键

快捷键功能
⌘+Shift+H快速聊天（全局唤起悬浮窗）
⌘+K命令面板（快速搜索/跳转）
⌘+,打开设置
⌘+N新建会话

对话页面

快捷键功能
Enter发送消息
Shift+Enter换行
⌘+F搜索对话
⌘+E导出当前会话
Esc停止生成

工作流画布

快捷键功能
⌘+S保存工作流
Delete删除选中节点
⌘+Z撤销
⌘+Shift+Z重做

导航

快捷键功能
⌘+1仪表盘
⌘+2对话
⌘+3Agent
⌘+B折叠/展开侧边栏

系统托盘

操作功能
左键点击托盘图标打开/聚焦主窗口
右键点击托盘图标弹出菜单（打开、快速聊天、日志、设置、退出）

命令面板

按 ⌘ + K 打开命令面板，支持以下操作：

导航命令

命令说明
概览跳转到 Dashboard
聊天跳转到聊天页
Agent跳转到 Agent 管理
任务跳转到任务调度
Skill跳转到技能管理
知识库跳转到知识库
记忆跳转到记忆管理
MCP跳转到 MCP 工具
IM 通道跳转到消息通道配置
日志跳转到实时日志
Canvas跳转到工作流画布
设置跳转到设置页

操作命令

命令说明
新建对话创建新会话并跳转到聊天页
切换主题在亮色/暗色模式之间切换

输入关键词可快速搜索过滤命令。

提示：macOS 使用 ⌘，Windows/Linux 对应 Ctrl。
